## 数据库监控

### 概述

此处使用的是 druid 数据库监控集成

### 使用说明

> 并不是所有的都需要数据库监控

添加监控注解:

```java
@EnableDruidMonitor // 添加监控
@SpringBootApplication
public class PortalWebApplication {
	public static void main(String[] args) {
		SpringApplication.run(PortalWebApplication.class, args);
	}
}
```
