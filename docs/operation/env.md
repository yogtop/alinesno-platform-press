# 部署进度

## 概要

基础环境完善及配置，为整个开发平台做基础，以环境搭建为主,为本地开发环境，
目前的服务器应该是不够的,也并不是代表所有的软件需要完善才可以进入下一步开发，比如 elk，前期搭建并一定是需要的，
比如说集群，也并不一定是一开始就是集群,
平台搭建一个人的话是比较长期，所以，以先投入组件构建为主，在完成第一步之后，再进一步完善，
即使在公司，也是如此建议，一步到位，也是比较难的，如果可以就更好。
